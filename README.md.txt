# Sistema de Cajero Automático - Java + Spring Boot

Este proyecto es una simulación de un sistema de cajero automático implementado en Java utilizando el framework Spring Boot. Permite a los usuarios realizar operaciones bancarias básicas como consultar saldo, retirar efectivo y ver movimientos.

## Tecnologías utilizadas

- Java 17
- Spring Boot 
- Maven
- Thymeleaf

## Funcionalidades principales


- Consulta de saldo
- Retiros de efectivo 
- Registro de movimientos
